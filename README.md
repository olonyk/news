### Dependencies
This project currently depends on the following packages:
1. docopt
2. hmmlearn
3. numpy
4. scikit-learn
5. scipy

These dependencies can be installed with:
```sh
$ sudo pip3 install numpy hmmlearn scikit-learn scipy docopt webstruct
```
